How It Works

1

Turn on Airport Security

Start the analysis by either starting the camera, which will use real camera footage, or by selecting a test image.

  • Start analysis
  • Observe results (image analysis results should be visible on the screen)
  • If a crowd result is there, then a security queue alert should be triggered
2

Open the Airline App

Open the Airline App from the header menu.

  • If a queue alert was triggered, then it should also be visible in the Airline App
3

Empty the queue

After triggering the queue alert, now either turn the camera to a non-congested view, or select a non-congested sampel image.

  • The queue alert should disappear in the Airport Security App
  • The queue alert should also disappear from the Airline App
